=head1 Synopsis

    data-demo list    - List open data sources
    data-demo import  - Import an open data source into a local db
    data-demo migrate <story>  - Apply a Migration Story

=head1 Description

L<Data::Demos> is set of tools for quickly creating demo databases and
applying migration stories to them.

It has the following concepts:

=over

=item C<data-demo>

The command line tool for interacting with Data::Demos.

=item Sources

Preconfigured ppen data sources

=item Databases

Various databases (tables) provided by the sources

=item Engines

Local databases engines to import databases into:

=over

=item * PostgreSQL

=item * MySQL

=item * SQLite

=item * more

=back

=item Schemas

Databases are imported adhering to certain schemas

=item Stories

Common tasks for database migrations.

=item Actions

Each Story is implemented as code so that it can be applied at any time.

=item Demos

Visual views of the data that can show off interesting things.

=back

=head1 Design

This project is primarily/initially a way to see various Perl database tools,
and specifically L<Class::DBIx::DeploymentHandler> in action. From there
people can add stories and actions for more and more useful things needed in
dealing with database migration.

=head1 Status

This project is currently in the concept and planning phase. Join the
